# create-dmg-action

Build fancy DMGs on your Github Actions workflows with just one simple action.

## Inputs

## `name`

**Required** Dmg file name. Default `"target"`.

## `srcdir`

**Required** Source dir path. Default `"srcdir"`.

## Example usage

```
uses: QQxiaoming/create-dmg-action@v1.0
with:
  name: 'target'
  srcdir: 'srcdir'
```
